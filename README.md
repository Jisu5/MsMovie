## Install

### Backend

```bash
$ cd backend
$ python install -r requirements.txt
$ python manage.py migrate
```

### Frontend

```sh
$ cd frontend
$ npm install
```

<br>

## Usage

### Backend

```bash
$ cd backend
$ python manage.py runserver
```

### Frontend

```sh
$ cd frontend
$ npm run serve
```

## Run tests

```sh
$ cd frontend
$ npm run test
```
